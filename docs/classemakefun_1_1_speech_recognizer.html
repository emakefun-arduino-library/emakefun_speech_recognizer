const String &  identification 
)

添加语音识别关键词

最多可以添加50个词，每个词最大长度为50个字节

参数
[in]index索引，范围 0 ~ 255，多个词可以共用一个索引，当在关键字触发( SpeechRecognizer::kKeywordTrigger )模式下，索引0会被当做关键词
[in]identification索引词，String类型，最大长度为50个字节
