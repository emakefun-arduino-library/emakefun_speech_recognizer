◆ IsActivated()

bool SpeechRecognizer::IsActivated ( )

查询是否已激活

每次通过该接口查询之后，系统会自动清除该标记位将其置为false，下次重新激活才会自动置为true； 该接口在自动激活模式( kAutoStart )下无效

返回值
true已经激活
false未激活
示例
button_trigger.ino , 以及 keyword_trigger.ino.
